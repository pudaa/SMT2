# SMT2 - 系统监控与待办事项管理工具

SMT2 是一个基于 Python 和 PySide6 开发的轻量级桌面应用程序，集成了系统监控和待办事项管理功能。该应用以系统托盘形式运行，占用资源极少，同时提供美观的信息展示界面。

## 功能特性

### 1. 时间进度监控

- 实时显示当前时间
- 日进度：当天已过去的时间百分比
- 周进度：本周已过去的天数百分比
- 月进度：本月已过去的天数百分比
- 年进度：本年已过去的时间百分比

### 2. 系统性能监控

- CPU 使用率
- 内存使用率
- GPU 使用率（占位实现）
- 电池电量状态

### 3. 待办事项管理

- 添加、编辑和删除待办事项
- 标记已完成的事项
- 自动提取任务标签
- 支持标签筛选功能

### 4. 用户界面特点

- 半透明效果界面
- 支持拖拽移动窗口
- 系统托盘图标支持
- 右键点击切换功能模式
- 双击关闭应用

## 使用方法

1. 运行应用后，会在桌面显示界面并在系统托盘区域显示图标
2. 主界面分为两个部分：
   - 上方为时间进度/性能监控面板
   - 下方为待办事项面板（默认隐藏，右键点击展开）
3. 界面操作：
   - 左键拖拽顶部区域移动窗口
   - 右键唤出待办事项面板
   - 双击顶部区域关闭应用
   - 右键点击系统托盘图标可切换性能监控模式

## 技术栈

- Python 3.12
- PySide6 (Qt for Python)
- psutil (系统信息获取)
- Nuitka (打包工具)

## 打包

使用 Nuitka 将应用打包为独立的可执行文件：

```powershell
python -m nuitka --onefile --windows-console-mode=disable --enable-plugin=pyside6 --follow-import-to=need --output-dir=output --windows-icon-from-ico=resources/tray.png --windows-uac-admin main.py
```

## 注意事项

- 该应用只有在Windows环境测试过，不保证对其他环境的支持
- 可能因为Windows的策略，打包后的可执行文件的置顶功能需要管理员权限支持
- 应用启动后会自动置顶显示，方便快速查看信息

## 隐私声明

本应用所有数据均存储在本地，不会上传任何用户数据到网络。待办事项等个人信息完全保留在用户设备上。
